# e.rest
RESTful API on Koa2(es7). Ready to use.

### Installation
~~~sh
$ npm install -g apidoc mocha
$ npm install
$ npm start
~~~

### Testing
~~~sh
$ npm start
$ npm test
~~~

or 

~~~sh
$ mocha --harmony start_server
~~~

### Generate docs
~~~sh
$ npm run docs
~~~
and go to link http://domain:3000/docs/
